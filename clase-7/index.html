01

Inspiración

Estética + Ritmo + Tono Narrativo
Referencias como input No copiar, traducir Criterio propio

La inspiración no es buscar “qué hacer”, sino definir “cómo se siente” lo que querés crear. Las referencias se transforman en prompts visuales y conceptuales que después traducimos en decisiones concretas.

02

Guión

Escena + Clip + Audio
Base del sistema Sinopsis al diálogo Pensado para IA

El guión es contenido pero también una estructura pensada para ser interpretada por modelos de generación. Tiene que ser claro, modular y funcional a la generación con las herramientas disponibles. Escribir para IA ≠ escritura tradicional.

03

Escenas

Estilo Visual + Iluminación + Intención
Unidad de generación Sistema modular Coherencia visual

Pensar en escenas permite dividir el proyecto en partes generables. Cada escena puede ser un prompt o muchos prompts distintos, pero tiene que mantener coherencia y entenderse como un todo. Del still al video, de los assets a la acción.

04

Personajes

Visual + Voz + Comportamiento
Identidad consistente Gestualidad Asset documentado

Un personaje es una combinación de elementos: apariencia, voz, forma de actuar. Cuanto más consistente es, más creíble resulta. Para eso documentamos al personaje como un “asset”: edad, tono, estilo visual, vestuario, comportamiento.

05

Voces

Capa emocional — Define interpretación
TTS generativo Une el sistema Múltiples versiones

El audio define cómo se interpreta todo lo que vemos. Una misma escena cambia completamente según la voz o la música. Pobá más de una versión de audio para el mismo video: podrías “salvar” alguna escena sin tener que rehacerla.

06

Escenarios

Contexto narrativo — Coherencia espacial
Refuerza emoción Define clima Escala y sensación

El escenario es parte de la historia y define el clima, la escala y la sensación del contenido. Dar contexto y a la vez dejar espacio al modelo es un buen camino. Después editamos lo que creamos necesario.

07

Audio

Capa emocional final del sistema
Voz en off Música generativa Diseño sonoro

El audio combina voz, música y diseño sonoro. La capa de audio es la que une todos los elementos visuales y les da sentido emocional. Es el último paso antes de la edición y puede redefinir una escena entera.

08

Edición

Donde todo se une y se redefine
Ritmo narrativo Corrección de inconsistencias Unidad final

La edición es el momento donde el contenido deja de ser piezas sueltas y se convierte en unidad. Aquí se define el ritmo narrativo, se corrigen las inconsistencias visuales y se integra todo el material generado en una pieza coherente.

Filosofía

Un Sistema Propio

El objetivo no es solo generar contenido, sino construir un flujo de trabajo replicable y mejorable.

Pensar Antes de Generar

La velocidad de la IA no significa saltear el proceso de pensamiento. La claridad del concepto y la estructura determinan la calidad del resultado. El modelo responde a lo que le das.

Organización es Calidad

Un pipeline ordenado es un pipeline controlable. Documentar assets, organizar referencias y estructurar el guión no es burocracia: es lo que permite iterar sin perder lo que ya funciona.

Replicable y Mejorable

El sistema que construís tiene que poder repetirse y perfeccionarse. Cada proyecto es un aprendizaje que mejora el siguiente. El valor está en el proceso, no solo en el resultado puntual.

Pensar en escenas permite dividir el proyecto en partes generables. No es posible describir todo pero si utilizamos los modelos a nuestro favor sabemos que dar contexto y a la vez dejar espacio al modelo es un buen camino.
